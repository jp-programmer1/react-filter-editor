### fieldComponent (React.Component)

```jsx
const FieldCustom = ({data, onChange, onEditField}) => (
  <input 
    type="text" 
    value={data.value} 
    onChange={(e) => onChange(e.target.value)}
  />
);

const options = [
  { name: "phone", label: "Cell Phone" fieldComponent: FieldCustom }
];
```
